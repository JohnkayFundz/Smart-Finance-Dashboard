Code of Conduct
Our Pledge
We pledge to make participation in the Smart Finance Dashboard project a welcoming, inclusive, and harassment‑free experience for everyone.
We are committed to fostering respect regardless of age, body size, disability, ethnicity, gender identity and expression, level of experience, nationality, personal appearance, race, religion, or sexual identity and orientation.

Purpose
The purpose of this Code of Conduct is to ensure a safe, collaborative, and professional environment for contributors and users.
It sets expectations for behavior and outlines processes for addressing misconduct.

Our Standards
Examples of positive behavior include:

Demonstrating respect and consideration in all interactions

Providing constructive feedback and accepting feedback gracefully

Helping others learn and grow

Focusing on what benefits the project and community

Examples of unacceptable behavior include:

Harassment, discrimination, or personal attacks

Use of insulting or derogatory language

Trolling or disruptive conduct

Publishing private information without consent

Any behavior that creates an unsafe or hostile environment

Enforcement
Project maintainers are responsible for clarifying and enforcing this Code of Conduct.
They may remove, edit, or reject contributions (comments, commits, code, issues, discussions) that violate this Code of Conduct.
Enforcement actions will be communicated when appropriate.

Scope
This Code of Conduct applies to all project spaces, including but not limited to:

GitHub Issues

Pull Requests

Discussions

Documentation

Community interactions related to this project

Reporting
If you experience or witness unacceptable behavior:

Please open a private issue or contact the project maintainer directly through GitHub.

Reports will be reviewed promptly, handled respectfully, and kept confidential.

Consequences
Violations may result in:

A warning

Temporary suspension from participation

Permanent removal from the project community

Acknowledgement
This Code of Conduct is inspired by the Contributor Covenant and other open‑source community guidelines.
By participating, you agree to uphold these standards and help create a welcoming and professional environment.
