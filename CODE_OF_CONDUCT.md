# Code of Conduct

## Our Pledge

We pledge to make participation in the Smart Finance Dashboard project a welcoming, inclusive, and harassment-free experience for everyone.

We are committed to fostering respect regardless of age, body size, disability, ethnicity, gender identity and expression, level of experience, nationality, personal appearance, race, religion, or sexual orientation.

---

## Purpose

The purpose of this Code of Conduct is to ensure a safe, collaborative, and professional environment for contributors and users.

It establishes expectations for respectful behavior and outlines how misconduct will be addressed.

---

## Our Standards

Examples of positive behavior include:

- Demonstrating respect and consideration in all interactions
- Providing constructive feedback and accepting feedback gracefully
- Helping others learn and grow
- Focusing on what benefits the project and community

Examples of unacceptable behavior include:

- Harassment, discrimination, or personal attacks
- Use of insulting or derogatory language
- Trolling or disruptive conduct
- Publishing private information without consent
- Any behavior that creates an unsafe or hostile environment

---

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct.

They may remove, edit, or reject contributions—including comments, commits, code, issues, pull requests, and discussions—that violate these standards.

Project maintainers will review reported incidents fairly and determine appropriate corrective actions based on the circumstances.

---

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub Issues
- Pull Requests
- Discussions
- Documentation
- Community interactions related to this project

---

## Reporting

If you experience or witness unacceptable behavior:

- Contact the project maintainer privately through GitHub or another published contact method.
- Avoid reporting Code of Conduct concerns in public GitHub issues whenever possible.
- Reports will be reviewed promptly, handled respectfully, and kept confidential whenever practical.

---

## Consequences

Violations may result in:

- A warning
- Temporary suspension from participation
- Permanent removal from the project community

---

## Acknowledgement

This Code of Conduct is inspired by the Contributor Covenant and other open-source community guidelines.

By participating in this project, you agree to uphold these standards and help create a welcoming, respectful, and collaborative community.
