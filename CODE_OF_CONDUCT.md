# Code of Conduct

## Our Pledge

We are committed to making participation in the Smart Finance Dashboard project a welcoming and respectful experience for everyone, regardless of experience level, background, or identity.

We pledge to foster an open, inclusive, and harassment-free community.

---

## Our Standards

Examples of behavior that contributes to a positive environment include:

- Being respectful and considerate.
- Providing constructive feedback.
- Accepting feedback gracefully.
- Helping others learn and grow.
- Focusing on what is best for the project and community.

Examples of unacceptable behavior include:

- Harassment or discrimination.
- Personal attacks or insulting language.
- Trolling or disruptive behavior.
- Publishing private information without permission.
- Any conduct that creates an unsafe environment.

---

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct.

They may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

---

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub Issues
- Pull Requests
- Discussions
- Documentation
- Community interactions related to this project

---

## Reporting

If you experience or witness unacceptable behavior, please open a private issue or contact the project maintainer through GitHub.

All reports will be reviewed promptly and handled respectfully.

---

Thank you for helping create a welcoming and professional open-source community.
