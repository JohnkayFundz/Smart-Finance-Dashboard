## 🚀 Quick Start

1. Clone the repository:

```bash
git clone https://github.com/JohnkayFundz/Smart-Finance-Dashboard.git
cd Smart-Finance-Dashboard
```

2. Open `index.html` in your preferred web browser.

3. Start tracking your finances.
